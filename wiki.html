Le Wikoucou

Le Wikoucou est tout simplement un jeu de mot entre wiki et coucou. En gros c'est un wiki qui parle des coucous. Il est divisé en deux catégories : une description détaillé des coucous et le coucou dans la culture.

Description

Cette page est une description détaillé des coucous, cela vous permettera d'en apprendre plus sur cette oiseau majestueux. Ma principale source est Wikipedia, je vais pas vous le cacher donc si il manque des infos soit c'est parce que j'ai pas trouver interessant de tous mettre ou que la page était pas complète. Si vous avez de nouvelle infos pour développer le site afin qu'il soit une référence pour les coucous, n'hésitez pas à me les faire parvenir en me contactant.

Culture

Cette page fait le tour des références aux coucous dans notre culture. Malheureusement je n'ai pas trouver beaucoup de choses à dire car le coucou est très peu représenter de manière générale. Donc si vous avez des informations à me les faire parvenir, contactez moi.
